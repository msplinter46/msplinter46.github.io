Legionella Pneumophila

Análise de homologias por Blast


As ferramentas de procura de homologias são bastante importantes e desta forma, requerem que os resultados obtidos para cada pesquisa sejam analisados procurando inferir pelas sequências homólogas as possíveis funções da sequência original.Este processo implica:
  • Analisar a lista de sequências homólogas
  • Identificar padrões consistentes ao nível da função desempenhada

O programa Basic Local Alignment Search Tool (BLAST) foi desenvolvido para realizar buscas comparando sequências biológicas primárias contra uma base de dados que contém uma quantidade gigantesca de informação.

A ferramenta BLAST é uma ferramenta muito utilizada dentro da bioinformática por ser uma ferramenta muito rápida e também pelo facto de existirem diversas variações do BLAST, cada uma com um objetivo específico.

Desenvolvemos scripts em BioPython para correr a ferramenta BLAST. Guardamos os resultados respetivos e criamos scripts para a sua análise semi-automática.
